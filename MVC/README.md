# MVC Projects

ASP.NET MVC applications built during the Capgemini .NET training program.

## Projects

| Project | Description |
|---------|-------------|
| `MVCapp1` | Introduction to MVC — controllers, views, routing |
| `MVCapp2` | Models, view models, form handling |
| `MVC_Net` | MVC with .NET — Razor views, layouts |
| `CRUDopperationinMVC` | Full CRUD operations using MVC |
| `mvcCodefirst` | Entity Framework Code-First approach |
| `EntityFramework` | Entity Framework integration with MVC |
| `MVCerrorHandeling` | Exception handling and error pages in MVC |
| `CollegeApplication` | College management application |
| `Enrollmentsystem` | Student enrollment system |
| `LibraryBookLendingSystem` | Library book lending management |

## Tech Stack

- ASP.NET MVC
- Entity Framework (Code-First)
- Razor Views
- SQL Server

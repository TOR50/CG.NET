# CG.NET — .NET & C# Learning Repository

A structured collection of C# and .NET exercises, projects, and assessments completed during the Capgemini training program.

## Repository Structure

| Folder | Topics Covered |
|--------|---------------|
| `day 1` | C# basics — variables, data types, conditionals, loops, switch |
| `day 2` | Mini-project — Finance Management System (OOP, classes) |
| `day 3` | Methods, exception handling, arrays |
| `day 4` | OOP — inheritance, polymorphism, abstract classes |
| `day 5` | Interfaces, namespaces, partial classes, enums |
| `day 6` | Structs, generics, static classes, nullable types |
| `day 7` | Arrays, collections (List, Dictionary, Queue, Stack, Hashtable) |
| `day 8` | Real-world console apps — MediSure Clinic, QuickMart |
| `day 9` | Advanced collections, dictionaries |
| `day 10` | Regular Expressions (Regex) |
| `day 11` | Tuples, anonymous types, LINQ |
| `day 12` | StringBuilder, memory management, GC |
| `day 13` | Delegates, events, callbacks, lambda expressions |
| `day 14` | File I/O — StreamReader, StreamWriter, BinaryReader/Writer |
| `day 15` | Serialization — JSON (`System.Text.Json`), XML |
| `day 16` | Reflection — Assembly, Type, MethodInfo, FieldInfo |
| `day 17` | Reflection continued, interface discovery |
| `MVC/` | ASP.NET MVC projects — CRUD, Entity Framework, Code-First, error handling |
| `Saturday_Assesment/` | Assessment projects — real-world console applications |
| `TopBrains/` | Assignments — C#, SQL, ADO.NET, LINQ, Entity Framework |
| `University Course Registration/` | Console-based course registration system |

## Tech Stack

- **Language:** C#
- **Framework:** .NET / ASP.NET MVC
- **ORM:** Entity Framework
- **Database:** SQL Server

## How to Run

```bash
# Console applications
dotnet run

# MVC projects
cd MVC/<project-name>
dotnet run
```
